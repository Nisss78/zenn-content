## はじめに

**「その格安ホテル、交通費と移動時間を含めても本当にお得ですか？」**

[STAYLINE](https://github.com/AbeShohei/STAYLINE)は、単なる「駅近」や「宿泊費の安さ」だけでなく、**目的地までの交通費・移動時間を含めた「トータルコスト（実質価格）」と「タイパ（時間対効果）」**でホテルを検索できるWebアプリケーションです。

公共交通オープンデータチャレンジ2025のエントリー作品として開発されました。

## 開発の背景

開発者のAbeShoheiさんは大の音楽好きで、頻繁にライブやコンサートに参加する中で以下の課題に直面していました。

- 人気アーティストのライブ時、会場周辺のホテルはすぐに満室または高騰
- 既存の旅行予約サイトは「エリア単位」の検索が主流で、「沿線軸」での横断検索が困難

そこで、不動産サイトでは当たり前の**「沿線・駅で探す」**機能にヒントを得て、「泊まる場所も沿線で探せれば選択肢が広がる」という着想に至りました。

### なぜ「東京メトロ」なのか？

1. **イベント会場を網羅**: 東京ドーム、日本武道館、渋谷のライブハウスなど主要会場の最寄り駅が集中
2. **ちょうどいい距離感**: 数駅離れるだけで静かでリーズナブルな宿泊エリアにアクセス可能

## 主な機能

### 1. 実質価格（トータルコスト）での比較

宿泊費だけでなく、目的地までの**往復電車運賃（IC/切符利用）を加算した合計金額**で比較できます。

「宿泊代は安いが遠くて交通費が高い」vs「少し高いが交通費が安い」の損得が一目で分かります。

### 2. タイパポイント（時間対効果）での比較

最寄り駅の最安ホテルを基準に、以下の計算式でタイパポイントを算出：

```
タイパポイント = お得額 ÷ (1 + 追加時間/10)
```

「少し遠いけど大幅に安い」「近いけど少し高い」など、時間と金額のトレードオフを数値化して比較できます。

### 3. 所要時間フィルター

移動時間（電車+徒歩）で5分〜90分（制限なし）まで絞り込み可能。

### 4. 移動情報の可視化

- **正確な移動時間**: 実際の列車時刻表データ + OSRMによる徒歩ルート計算
- **終電・始発チェック**: 宿泊日から逆算して自動表示

### 5. 直感的なUI/UX

- **Neumorphism Design**: ソフトで触りたくなる質感
- **キーボード操作を排除**: 片手でストレスなく操作可能
- **レスポンシブ対応**: スマートフォンでも最適化

## 使用データ・API

### 公共交通オープンデータセンター (ODPT)

- `odpt:Railway` (路線情報)
- `odpt:Station` (駅情報)
- `odpt:RailwayFare` (運賃情報：IC/切符)
- `odpt:TrainTimetable` (列車時刻表)
- `odpt:StationTimetable` (駅時刻表)

### 楽天ウェブサービス

楽天トラベル施設検索API (VacantHotelSearch) を使用して、ホテルの空室情報・価格・画像・レビュー情報を取得。

### OpenStreetMap / OSRM

駅からホテルまでの正確な徒歩ルート計算に使用。

## 技術的な工夫

### 多角的なデータ結合

ODPTが提供する「運賃」「時刻表」「駅情報」と、楽天トラベルの「宿泊施設情報」という異なる性質のビッグデータを、位置情報と駅マスタをキーに動的に結合。単純な距離検索ではなく、「移動コスト」という複合的な指標でのリアルタイムソートを実現しました。

### リアルな「徒歩」へのこだわり

駅からホテルまでの距離は、**直線距離ではなくOSRM (Open Source Routing Machine)** を用いて実際の道路ネットワークに基づいた徒歩ルートを計算。「直線距離では近いが、線路を渡れず実際は遠回りが必要」といったミスマッチを防ぎます。

## 社会的意義

### 公共交通の活性化

- **交通空白の解消**: マイナーな駅でも「実は目的地まで乗り換えなしで15分」という価値を可視化
- **Win-Winな経済循環**: 宿泊費の節約分が交通費として鉄道事業者に還元される仕組み

### オーバーツーリズムの緩和

一極集中しがちな宿泊需要を沿線全体に分散させることで、混雑緩和と沿線地域の経済活性化に寄与。

## 今後の展望

この「沿線・駅単位での横断的検索」モデルは、**地方の観光・地域活性化**にも応用可能です。

- 京都や鎌倉などの人気観光地でのオーバーツーリズム解消
- 「電車で1本、20分離れた場所」にある穴場ホテルの提案
- 隠れた名所と宿泊のセット提案による地域経済活性化

## セットアップ方法

```bash
# クローン
git clone https://github.com/AbeShohei/STAYLINE.git
cd STAYLINE

# インストール
npm install

# 環境変数の設定 (.env.local)
VITE_ODPT_API_KEY=your_odpt_api_key
VITE_RAKUTEN_APP_ID=your_rakuten_app_id

# 開発サーバー起動
npm run dev
```
